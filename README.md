# 🛍️ ShopSphere CRUD App

A modern Flutter CRUD application built using **Bloc State Management** and **Dio API integration**.

This project demonstrates full **Create, Read, Update, Delete (CRUD)** operations with clean architecture, responsive UI design, and modern Flutter development practices.

---

# 📱 Features

✅ Fetch Products from API  
✅ Add New Product  
✅ Update Existing Product  
✅ Delete Product  
✅ Search Products  
✅ Product Details Screen  
✅ Pull to Refresh  
✅ Shimmer Loading Animation  
✅ Responsive UI using ScreenUtil  
✅ Error Handling & Loading States  
✅ Clean Project Structure  
✅ Bloc State Management  
✅ Dio Network Requests

# 📌 Important Note

The project uses the DummyJSON API for demonstration purposes.

Some API endpoints simulate CRUD operations without permanently modifying server data.

To provide a smooth user experience, local state updates are handled inside Bloc.
